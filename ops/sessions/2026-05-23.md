## Diff summary
Adds a `shard.yml` branch to `detectStack`. Pattern is a near-clone of
the pubspec.yaml block already in place, so reviewers can verify by
diffing the two. The nested-version test guards against the same trap
pubspec had — a `version:` under `dependencies:` (Crystal source-spec
syntax for a dep that pins to a version range) is one of the easiest
ways for the regex to mis-fire. Quote/comment stripping reuses the
exact character class `[^'"\s#]+` from pubspec so behavior matches.
